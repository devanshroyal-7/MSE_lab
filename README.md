# MSE_lab
MATLAB codebase for 24-452 Mechanical Systems Experimentation course at CMU.

MSE_PLANT is the slx file that is run in Desktop Real-time mode along with EMB Spring-Mass-Damper (SMD) kit from [ROBOTS5](https://www.robots5.com/)

MSE_App.mlapp is the MATLAB app which is in development. It will be the interface with which the students will interact with the SMD. 

There are are 5 labs, each lab may or may not have its own file, but will have the bare-minimum code to test the functionality of the SMD system.
